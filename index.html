HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The footer contains the author, copyright, contact, sitemap, navigation.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

CSS

  • 3 ways to style with CSS: inline (specific tag or element), internal CSS stylesheet, external CSS style sheet.
  • img { display: block; } causes image to start on a new line.

Git

  • git status = checks what branch we are currently on.
  • git checkout -b branch name = creates new branch and switches to it.
  • The Git Flow Process:
    1. git add -A
    2. git commit -m "message"
    3. git pull origin main
    4. git push origin feature/{branch-name}
    5. go to Github and create new PR
    6. merge changes to main branch
    7. git checkout main
    8. git pull origin main to sync Git and Github

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • An array is a single variable that is used to hold a group of data. They are useful when we want to perform a task that involves multiple pieces of data at once.
  • Index numbers are how we identify a location within the array
  • For loops are made of 3 parts: starting point, condition and iteration. For example: for (var x = 0; x < array.length; x++);
  • Functions are a set of instructions that tell the computer how to perform a task. They differ from for loops because they don't automatically run. They are made of 2 parts: define function and call function. Example: function name();.
  • Math.floor() and Math.random() can be used to generate a random number.
